# Telecom Customer Churn Prediction

## Overview
This project predicts customer churn for a telecommunications company using Machine Learning and Business Analytics techniques.

## Objectives
- Predict customer churn
- Identify high-risk customers
- Discover key churn drivers
- Support customer retention strategies

## Dataset
- Client.csv
- Record.csv

## Workflow
1. Data Cleaning
2. Exploratory Data Analysis (EDA)
3. Feature Engineering
4. Model Training
5. Model Evaluation
6. SHAP Explainability
7. Business Recommendations

## Machine Learning Models
- Logistic Regression
- Random Forest
- XGBoost
- LightGBM (Best Model)

## Best Performance
- Best Model: LightGBM
- ROC-AUC ≈ 0.697
